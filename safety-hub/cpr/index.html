심폐소생술(CPR) 핵심

기억은 짧게: 확인 → 요청 → 압박 → AED

1) 반응 확인

  • 어깨 두드리며 “괜찮으세요?”
  • 호흡 확인(10초 이내)
  • 호흡 없거나 비정상이면 즉시 CPR

2) 119 + AED 요청

  • 119 신고(스피커폰 권장)
  • 주변에 AED 가져오라고 요청
  • 정확한 위치(층/문앞) 전달

3) 흉부 압박

  • 가슴 중앙(흉골) 손깍지
  • 빠르고 강하게, 완전 이완
  • 가능하면 30:2 반복(훈련 시)

AED 사용

전원 ON → 패드 부착 → “떨어지세요” → 음성 안내에 따르기.

현장 체크 허브로

※ 실제 CPR 기준은 공식 교육(대한적십자/소방 등) 안내를 따르세요.
